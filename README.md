# Python + Data Engineering Learning Path

This repository contains a guided, notebook-first curriculum for Python fundamentals and Data Engineering interview preparation.

## Environment Setup

1. Create a virtual environment:
   - `python3 -m venv .venv`
2. Activate it:
   - macOS/Linux: `source .venv/bin/activate`
3. Install dependencies:
   - `pip install -r requirements.txt`
